Projects

ASL Handsign Machine Learning Model

This project aims to identify American Sign Language (ASL) fingerspelling images using two machine learning models: a Convolutional Neural Network (CNN) and a Support Vector Machine (SVM). The project explores the strengths and weaknesses of both models in classifying hand signs into the 26 letters of the alphabet with high accuracy.

The CNN model was able to achieve 99.5% accuracy by automatically learning features from raw image data, while the SVM model, using Histogram of Gradients (HOG) feature extraction, achieved 100% accuracy. The dataset consisted of 87,000 ASL fingerspelling images, and various techniques like dimensionality reduction were employed to optimize performance.

This project demonstrates the potential of machine learning models in solving complex computer vision problems, with practical applications in the field of accessibility for deaf and hard-of-hearing individuals.
